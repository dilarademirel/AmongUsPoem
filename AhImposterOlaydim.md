# Ah imposter olaydım

```
Ah imposter olaydım,
Seni yalnız bulaydım.
Birlikte görev yapıp,
Güvenin kazanaydım.

Ah imposter olaydım,
Ventten vente atlayaydım.
Önüme geleni kesip,
Self raporlayaydım.

Ah imposter olaydım,
Sabotajlar yapaydım.
Kapıları kapatıp,
Double kill alaydım.

Ah imposter olaydım,
Reaktörde dolanaydım.
Kameraları izlemeyip,
Oyundan atılaydım.
```
-- By **Murat Erdem**
